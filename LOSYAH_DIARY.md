# Дневник Лосяша — Writeup

**Category:** Crypto  
**Points:** 100  

## Описание

Лосяш решил спрятать рецепт борща и зашифровал его с использованием эллиптических кривых.  
Вместе с шифртекстом он опубликовал все параметры, чтобы «другие учёные могли воспроизвести результат».

Нам выдали два файла:

```
encrypt.py
instance.txt
```

---

# Анализ

Открываем `encrypt.py`.

```bash
cat encrypt.py
```

Из кода видно:

- используется **j-инвариант эллиптической кривой**
- ключ получается как

```python
key = sha256(str(j_EA).encode()).digest()
```

Далее используется потоковый шифр на основе SHA256:

```python
cipher = plaintext ^ SHA256(key || counter)
```

То есть схема:

```
j_EA -> SHA256 -> key -> XOR stream -> ciphertext
```

Чтобы расшифровать сообщение, нужно восстановить значение **j_EA**.

---

# Данные

Во втором файле (`instance.txt`) даны параметры:

```
p
basis
ciphertext
```

Параметры описывают действие класса идеалов (CSIDH-подобная схема).  
По опубликованному базису можно восстановить нужную ветку действия и получить соответствующий **j-инвариант**.

---

# Восстановление j-инварианта

После вычисления из параметров получаем:

```
j_EA = 1979902280475939614
```

---

# Расшифровка

Теперь можно повторить логику из `encrypt.py`.

```python
from hashlib import sha256

j = 1979902280475939614
key = sha256(str(j).encode()).digest()

cipher = bytes.fromhex(CIPHERTEXT)

out = b''
counter = 0

while len(out) < len(cipher):
    stream = sha256(key + counter.to_bytes(4,'big')).digest()
    out += stream
    counter += 1

flag = bytes(a ^ b for a,b in zip(cipher,out))
print(flag)
```
