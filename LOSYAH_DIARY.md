# Че дано

Сначала смотрим код шифрования.

```
 encrypt.py
```

Из кода видно, что ключ строится из **j-инварианта эллиптической кривой**:

```python
key = sha256(str(j_EA).encode()).digest()
```

Дальше используется потоковый шифр: генерируется поток через SHA256 со счётчиком и XOR'ится с сообщением.

Схема получается такая:

```
j_EA -> SHA256 -> key -> XOR stream -> ciphertext
```

Следовательно, если узнать значение `j_EA`, можно восстановить ключ и расшифровать сообщение.

# Берем логику и подставляем че нашли

Теперь повторяем логику из `encrypt.py`.

```python
from hashlib import sha256

j = 1979902280475939614
key = sha256(str(j).encode()).digest()

cipher = bytes.fromhex("76c8444dead7ed9374dab81a799a18fc0b3e30a65aebfe2da61eb4ad54e38316")

out = b''
counter = 0

while len(out) < len(cipher):
    stream = sha256(key + counter.to_bytes(4,'big')).digest()
    out += stream
    counter += 1

flag = bytes(a ^ b for a,b in zip(cipher,out))
print(flag)
```
